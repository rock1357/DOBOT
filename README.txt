# DOBot

This is a Qt-based C++ bot simulation project using CMake.  
It includes a multithreaded state machine and a GameController for automating mouse-controlled bot behavior.

## Features

- Qt GUI with start button
- State machine running in a background thread
- Mouse control via native OS APIs
- CMake-based cross-platform setup
